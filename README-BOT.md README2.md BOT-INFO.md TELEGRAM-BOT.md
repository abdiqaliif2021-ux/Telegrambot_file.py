# Telegram Bot Mobile

Bot-kan waxa uu u diraa fariin channel-ka Telegram.  
Waxaa la isticmaalaa Python iyo python-telegram-bot library.  

## Files
- `bot.py` → code-ka bot-ka
- `requirements.txt` → dependency Python
- `README-BOT.md` → sharaxaad repository-ga

## Usage
1. Ku dar BOT_TOKEN iyo CHANNEL_ID ee Render .env
2. Deploy bot.py on Render
3. Bot-ka waa online 24/7
